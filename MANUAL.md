## 1. はじめに

将棋部内戦マッチングシステムは、部内戦の対戦組み合わせを自動生成し、結果・成績を管理するWebアプリです。

**主な機能**
- 選手の登録・管理（ランク・学年付き）
- 自動マッチング（ランダム方式・スイスドロー方式）
- 先手/後手の自動均等化
- 対戦結果の入力と成績表の自動計算
- 対戦表・成績表のPDF出力
- 複数デバイスでのリアルタイム共有（部屋コードで共有）

## 2. アカウントと部屋

### アカウント登録・ログイン

初回利用時は「新規登録」タブからアカウントを作成します。

| 項目 | 内容 |
|------|------|
| ユーザー名 | 半角英数字推奨 |
| パスワード | 任意の文字列 |

登録後は「ログイン」タブからサインインします。パスワードは変更可能です（アカウント設定から）。

### 部屋とは

「部屋」は大会ごとのデータ管理単位です。部屋ごとに選手・対戦・成績が完全に分離されます。

**部屋の作成**
1. ログイン後、「新規作成」ボタンをクリック
2. ランダムな6文字の部屋コード（例：`AB3K9M`）が生成される
3. そのコードを参加者に共有する

**部屋への参加**
1. 「参加」ボタンをクリック
2. 共有された部屋コードを入力
3. 参加完了（他の参加者の操作がリアルタイムで反映される）

**URLによる直接アクセス**
`?room=AB3K9M` のURLを共有すると、ログイン後に自動で部屋に入室できます。

**最近使った部屋**
以前に使用した部屋は履歴として一覧表示されます（アカウントごとに最大5件）。

> **ヒント:** 毎年の部内戦は部屋を分けて管理することで、過去のデータを保持できます。

## 3. 選手管理

「選手管理」タブで選手の追加・編集・削除を行います。

### 選手の登録

| 項目 | 内容 |
|------|------|
| 名前 | 選手の氏名 |
| 学年 | 1〜8年 |
| ランク | S / A / B / C（強い順） |

フォームに入力して「追加」ボタンをクリックします。

### ランクの目安

| ランク | 目安 |
|--------|------|
| S | 部内トップクラス |
| A | 上位層 |
| B | 中位層 |
| C | 初級〜中級 |

### 選手一覧

登録した選手は学年順・ランク順で一覧表示されます。
- ✏️ アイコン：選手情報の編集
- 🗑 アイコン：選手の削除

> **注意:** 対戦表生成後に選手を削除すると、対戦データが正常に表示されない場合があります。選手の追加・削除はマッチング前に行うことを推奨します。

## 4. マッチング生成

「マッチング」タブで対戦組み合わせを生成します。

### 設定項目

#### 方式

| 方式 | 説明 |
|------|------|
| **ランダム方式** | 全ラウンドを一括で生成。制約の範囲内でランダムに組み合わせる。 |
| **スイスドロー** | 成績に応じて1ラウンドずつ生成。上位同士・下位同士が対戦しやすくなる。 |

#### ラウンド数

3〜10回戦まで設定できます。

#### 先手・後手の割り当て

| 設定 | 説明 |
|------|------|
| **指定あり（先後均等化）** | 各選手の先手・後手の回数が均等になるよう自動調整。PDF対戦表には「先手」「後手」列が表示される。 |
| **指定なし** | 先後の割り当てなし。PDF対戦表には「選手①」「選手②」列が表示される。 |

> **先後均等化の仕組み:** 対戦ごとに双方の先手出場回数を比較し、少ない方を先手に配置します。これにより、大会を通じた先後のバランスが保たれます。

#### 同学年対決の回避

同じ学年同士の対戦が組まれにくくなります。「無効」にするとまったく考慮しません。

| 設定 | 効果 |
|------|------|
| 無効 | 同学年対決を気にしない |
| 低〜最高 | 高いほど同学年対決を避ける |

#### ランク帯の均等化

各選手が様々なランクの相手と対戦するよう調整します。「無効」にするとランクを考慮しません。

| 設定 | 効果 |
|------|------|
| 無効 | ランク分布を気にしない |
| 低〜最高 | 高いほど対戦ランクが分散する |

> **ランク均等化の目的:** S・A・B・Cそれぞれのランクの相手と満遍なく対戦できるよう誘導します。例えば、すでにCランクと多く対戦した選手は、次のラウンドではCランク以外と組まれやすくなります。

### 再戦回避について

同じ2人が同じ大会で2度対戦することは**絶対に起こりません。** これはどのラウンドでも保証されています。

> ただし、参加人数が少ないか、ラウンド数が多い場合は再戦なしで組めない場合があります。その際はエラーメッセージが表示されます。

### 不戦勝（空き手合い）

参加人数が奇数の場合、毎ラウンド1人が不戦勝になります。

- 不戦勝に選ばれた選手はポイント +2（勝利と同等）
- 不戦勝の回数が最も少ない選手が優先的に選ばれる

### スイスドローの進め方

1. 「第1回戦を生成」をクリック
2. 結果入力タブで全対局の結果を入力
3. マッチングタブに戻り「第2回戦を生成」をクリック
4. 以降、同様に繰り返す

> 前のラウンドの全結果が入力されていないと、次のラウンドは生成できません。

### 手動編集モード

生成した対戦組み合わせを手動で変更できます。

**編集の手順**
1. 「手動編集」ボタンをクリック（編集モードに入る）
2. 入れ替えたい選手名をタップ → 選手がハイライト表示
3. 入れ替え先の選手名をタップ → 2人が入れ替わる
4. 「⇄」ボタン → 同じ対局内で先手・後手を入れ替える
5. 「編集完了」で保存 / 「キャンセル」で元に戻す

> **注意:** 入れ替えた対局の結果は自動的にリセットされます。

### 表示オプション

マッチングタブ・結果入力タブの上部にある「学年」「ランク」チップをクリックして、一覧での表示・非表示を切り替えられます。この設定はPDF出力にも反映されます。

## 5. 結果入力

「結果入力」タブで各対局の勝敗を記録します。

### 結果の入力方法

各対局に3つのボタンが表示されます。

| ボタン | 意味 |
|--------|------|
| 左の ○ | 左側の選手（先手/選手①）の勝ち |
| △ | 引き分け |
| 右の ○ | 右側の選手（後手/選手②）の勝ち |

選択すると自動保存され、✕ボタンで結果をリセットできます。

### 進行状況

各ラウンドカードに「2/3 対局完了」のように進行状況が表示されます。

## 6. 成績順位表

「成績表」タブで現在の順位と成績を確認できます。

### 順位の決定方法

以下の優先順位で順位を決定します。

1. **ポイント（Pt）** の高い順
2. 同Pt → **（勝利数 + 不戦勝数）** の多い順
3. さらに同じ → **勝率** の高い順

### ポイントの計算

| 結果 | 獲得Pt |
|------|--------|
| 対局勝利 | +2 Pt |
| 不戦勝（BYE） | +2 Pt |
| 引き分け | +1 Pt |
| 敗北 | +0 Pt |

「Pt算出方法 ℹ」ボタンからこのルールをいつでも確認できます。

### 表の見方

| 列 | 意味 |
|----|------|
| 順位 | 上記ルールによる順位（同点は同順位） |
| 名前 | クリックで個人成績モーダルを開く |
| 学年 | 選手の学年 |
| 勝 | 対局での勝利数（不戦勝は含まない） |
| 敗 | 敗北数 |
| 分 | 引き分け数 |
| 不戦勝 | BYEの回数 |
| 勝率 | 勝 ÷（勝 + 敗 + 分）× 100（不戦勝は含まない） |
| Pt | 合計ポイント |

> **Pt列の非表示:** 「Pt 表示中」ボタンをクリックするとPt列を隠せます。順位や集計結果を発表する際などに便利です。

## 7. 個人成績

成績表の選手名をクリックすると、個人の詳細成績モーダルが開きます。

### 表示内容

- **サマリー:** 勝・敗・分・不戦勝・勝率・Pt
- **ランク別戦績:** S・A・B・Cランクの相手との勝敗内訳と棒グラフ
- **対戦履歴:** 回戦ごとの対戦相手・先後・結果

## 8. PDF出力

### 対戦表のPDF出力

マッチングタブ（または結果入力タブ）の「📄 PDF出力」ボタンをクリックします。

- 全ラウンドの組み合わせが表示される
- 先後指定あり → 「先手」「後手」列で表示
- 先後指定なし → 「選手①」「選手②」列で表示
- 「学年」「ランク」の表示オプションが反映される

### 成績表のPDF出力

成績表タブの「📄 成績表PDF出力」ボタンをクリックします。

- 現時点の成績順位表が出力される
- 順位・名前・学年・勝敗・不戦勝・勝率・Ptを含む

### PDFの保存方法

ボタンをクリックすると新しいタブが開き、印刷ダイアログが自動起動します。

| ブラウザ | 操作 |
|----------|------|
| Chrome / Edge | 送信先 → 「PDFに保存」を選択 |
| Safari | 左下「PDF」→「PDFとして保存」 |

> **ポップアップがブロックされた場合:** ブラウザのアドレスバー右端に表示されるアイコンから、ポップアップを許可してください。

## 9. よくある質問

**Q. マッチングの生成に失敗しましたと表示される**
A. 参加人数が少なくラウンド数が多い場合、再戦なしで組み合わせを作れないことがあります。ラウンド数を減らすか、参加人数を増やしてください。

**Q. 途中で選手を追加できますか？**
A. 技術的には可能ですが、既に生成された対戦表には反映されません。選手の追加はマッチング前に行うことを推奨します。

**Q. 部屋のデータを完全にリセットしたい**
A. 選手管理タブの「全データリセット」ボタンで、その部屋の全選手・対戦・結果を削除できます。

**Q. 複数のデバイスで同時に操作できますか？**
A. できます。同じ部屋コードで入室したデバイス同士はリアルタイムで同期されます。ただし、複数人が同時に対戦表を編集すると競合が起こる場合があります。

**Q. オフラインでも使えますか？**
A. 基本的な閲覧・入力は端末内（LocalStorage）に保存されるためオフラインでも動作します。ただしクラウド同期は、オンラインに戻った際に反映されます。

**Q. パスワードを変更したい**
A. 画面下部の「パスワードを変更」セクションから変更できます。現在のパスワードの確認が必要です。
